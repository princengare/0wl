automatic updates

0wl 0.1.4 is approved on Mozilla Add-ons. Permanent Firefox installs can update through AMO or a signed self-hosted release channel.

firefox add-ons listing

AMO path

0wl is listed on addons.mozilla.org. After each new listed version is reviewed and approved, Firefox handles extension updates for users.

self-hosted path

UPDATE_BASE_URL="https://downloads.example.org/0wl" XPI_FILE="0wl-0.1.4.xpi" npm run updates:manifest

Host the signed XPI and generated updates.json over HTTPS, then ship a real gecko.update_url.

Do not ship a placeholder update URL.

update safety

  • Lifecycle metadata is recorded locally.
  • Settings are validated and migrated forward.
  • Dynamic block and limit rules are rebuilt from settings.
  • Stale active sessions are not credited with Firefox downtime.
