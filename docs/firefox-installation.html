For manual loading, run npm run build:firefox, open about:debugging#/runtime/this-firefox, and select .output/firefox-mv3/manifest.json.

npm run release:prepare
npm run sign:firefox

Maintainers can still create signed release packages. The recommended persistent path for normal users is the Firefox Add-ons listing.
