local-first website focus tracker

0wl is a standalone local-first extension for tracking active website time, blocking distracting domains, scheduling blocks and limits, and reviewing terminal-style history charts, local behavioral insights, scheduled friction, and local data controls. Version 0.1.8 hardens update safety, repairs impossible zero-limit/private tracking artifacts, removes impossible overlapping History buckets, restores regular-window live History and Today visibility, and keeps blocked/time-limit interstitials inside one viewport from one shared codebase. No accounts. No backend. No cloud sync.

Current features

usage tracking Tracks active HTTP/HTTPS domains only while Firefox is focused and the system is active.
history charts Today and yesterday use hourly bars. This week uses calendar-week bars and an average line.
terminal ui controls Custom dropdowns, bracket checkboxes, one-line dashboard tabs, and bundled JetBrains Mono keep the UI consistent.
scheduled blocking Block websites always or only on selected local days and times, including overnight windows.
scheduled limits Apply daily time limits always or only during scheduled windows, with a temporary bypass page.
vision insights Local site categories, compressed distraction pathways, drift summaries, block-gated evasion signals, recovery estimates, and recommendations.
data control View local data status, export a JSON backup, import backups, set retention, and clear specific local data after confirmation.
local storage Uses IndexedDB and browser.storage.local. No telemetry, accounts, or cloud database.
private browsing controls Private/incognito tracking is off by default, scope-aware where browser access is granted, and linked to the privacy policy from the enable popup.
browser-specific builds WXT builds Firefox, Chrome, Edge, Opera, and Safari web-extension assets from the shared source tree.
future local sync A future milestone is local-device syncing across the browsers installed on the same computer without accounts or cloud sync.
